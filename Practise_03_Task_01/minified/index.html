Язык разметки Marcdown

Общие правила

Язык Marcdown позволяет описать структуру текстового документа без использования трудночитаемых языков разметки вроде HTML

Пример

<strong>Важный текст<strong>

Заголовки

Для указания заголовка определенного уровня в Marcdown нужно перед текстом заголовка поставить набор знаков #.

Вы можете использовать HTML тэги внутри языка Marcdown.

Пример:

## Заголовок второго уровня
    ###### Заголовок шестого уровня

При конвертировании в HTML получим следующее:

Заголовок второго уровня

Заголовок шестого уровня

Списки

Для создания списка перед каждым элементом списка нужно поставить символ *

Пример:

* первый элемент списка 
    * второй элемент списка
    * третий элемент списка

При конвертировании в HTML получим следующее:

  • первый элемент списка
  • второй элемент списка
  • третий элемент списка

Для создания нумерованного списка перед каждым элементом списка нужно поставить число.

1. первый элемент списка
    2. второй элемент списка
    3. третий элемент списка

При конвертировании в HTML получим следующее:

  1. первый элемент списка
  2. второй элемент списка
  3. третий элемент списка

Для создания списка определений после определяемого слова нужно поставить символ :

Пример:

kramdown
    : Альернативный вариант языка Marcdown
    
    Maruku
    :     Другой конвертор языка Marcdown

При конвертированиив в HTML получим следующее:

kramdown
Альтернативный вариант языка Marcdown
Maruku
Другой конвертор языка Marcdown

Ссылки

Для создания ссылок нужно использовать следующий синтаксис:

[текст ссылки](https://google.com)

При конвертировании в HTML получим следующее:

текст ссылки

Сильная значимость и акцентирование

Для указания важного текста в документе нужно использовать два символа ** или __

**важный текст**
    __важный текст__

При конвертировании в HTML получим следующее:

важный текст важный текст

Для указания важного акуента на слово или фразу нужно использовать один символа * или _

*акцентирование*
    _акцентирование_

При конвертировании в HTML получим следующее:

акцентирование акцентирование

Цитаты

Для цитирования, перед каждой строкой цитаты нужно поставить знак >

>

Пример:

>Words can be like X-rays, if you use them properly—
    they’ll go through anything. You read and you’re pierced.

При конвертировании в HTML получим следующее:

Words can be like X-rays, if you use them properly— they’ll go through anything. You read and you’re pierced.
